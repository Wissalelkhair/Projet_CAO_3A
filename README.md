Afin de réaliser l’assemblage des moteurs, il est maintenant nécessaire de déterminer et de choisir les
roulements nécessaires à son bon fonctionnement. Pour ce faire, nous avons utilisé le site catalogue
« traceparts » qui regroupe un très grand nombre de références de roulements mais également tout
type de composants mécaniques ainsi que leurs fichier CAO.
Suite aux analyses éléments finis que nous avons réalisés sur les arbres monoblocs et assemblés
correspondant aux différents couples, nous avons pu déterminer les forces exercées sur l’arbre durant
son utilisation. Nous avons donc pu observer que l’arbre n’est soumis à aucun effort axial, mais
seulement à des efforts radiaux. C’est pourquoi les roulements à rouleaux sont le plus adapté dans cette
situation.
